Irish Housing Crisis Project

The subject of this project is the housing crisis in Ireland and the UK.
Analysis comtains the workflow from data ingestion, EDA, sentiment analysis to predictive time series modelling.

  • Are newly built houses keeping up with population growth in UK & Ireland?
  • When did people start to talk about the housing crisis?
  • Can we extract sentiment from tweets around the housing crisis?
  • Will current trends in houses being built be able to accomodate the population over time?

    Forecasting Methodologies in a Big Data Context

    This project examines the practical application of neural networks in the context of big data analytics, with a particular focus on improving time series analysis. With a focus on financial forecasting for a large retail business, the project compares advanced processing tools, machine learning technologies, and neural networks to determine the most effective ways to improve forecast accuracy. We presents a comparison of various time series analysis methods, including classical statistical approaches, machine learning models, and deep learning neural networks, highlighting their strengths and weaknesses in handling datasets with slightly different compositions.

    Work is carried out leveraging Hadoop, Spark, Keras, Tensorflow, Prophet and Scikit Learn. EDA performed to analyse time series, forecasting models developed and evaluated.

    Instacart Cluster Analysis

    This project aims to develop a set of customer segments to facilitate more effective conversion through better targeting with promotional material or offerings. “Market segmentation is to divide a market into smaller groups of buyers with distinct needs, characteristics, or behaviors who might require separate products or marketing mixes.”
    To develop this solution, the Instacart Historical sample shopping data (https://www.instacart.com/datasets/grocery-shopping-2017) will be used simulate typical retail data. This dataset was released initially as part of a Kaggle competition to develop robust association rules but in this case will be used to inform behaviour to identify customer segments.
